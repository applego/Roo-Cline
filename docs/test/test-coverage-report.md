# テストカバレッジレポート

## 概要

このドキュメントは、WebSocketサーバーの実装に対するテストカバレッジの結果をまとめたものです。

## テスト実行方法

```bash
# テストの実行とカバレッジレポートの生成
cd server
pnpm test:coverage
```

## 注意事項

1. テストの実行には以下の環境が必要です：

    - Node.js 18以上
    - pnpm 8以上

2. テスト実行時の注意点：
    - テスト用のポート（8080）が空いていることを確認
    - 十分なメモリ（最低2GB）が利用可能であること
